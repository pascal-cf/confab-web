## Why Confabulous?

<CardGrid>
  <Card title="Self-host or fully managed" icon="rocket">
    Run the full stack on your own infrastructure with Docker Compose, or use the free managed instance at [confabulous.dev](https://confabulous.dev).
  </Card>
  <Card title="Multi-provider" icon="puzzle">
    First-class support for Claude Code and OpenAI Codex. One dashboard for both.
  </Card>
  <Card title="Analytics & insights" icon="bars">
    Cost tracking, Smart Recaps, TILs, personal Trends, and org-wide analytics across every session you and your team run.
  </Card>
  <Card title="No vendor lock-in" icon="open-book">
    MIT-licensed, open source. Your data stays in your Postgres and your object store.
  </Card>
</CardGrid>
