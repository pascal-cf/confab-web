The **Sessions** page lists every session you own or have access to, with filters for provider, repo, owner, date range, and more.

## Session viewer

Clicking a session opens a three-pane view:

- **Summary** — analytics cards (cost, tokens, tools, conversation shape).
- **Transcript** — the full conversation, rendered with collapsible tool calls.
- **Smart Recap** — AI-generated summary of what the session accomplished.

## Filters

- **Provider** — Claude Code, Codex.
- **Repo** — forks roll up to their upstream root automatically.
- **Owner** — who ran the session.
- **Date range**.
- **Free-text search** — full-text over transcripts.
